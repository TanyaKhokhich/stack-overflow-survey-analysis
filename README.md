# Stack Overflow Developer Survey Analysis

## Overview

This project explores the Stack Overflow Developer Survey dataset using Python and Pandas. The goal was to analyze developer demographics, technologies, and salary trends through exploratory data analysis.

* Python
* Pandas
* Data Analysis

* Cleaned and filtered data
* Handled missing values
* Calculated statistical metrics
* Performed data aggregation and grouping
* Identified trends and patterns in developer data
